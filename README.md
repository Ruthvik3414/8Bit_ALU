# 8-bit ALU in Verilog

This project implements an 8-bit Arithmetic Logic Unit (ALU) using Verilog HDL.

## Operations Supported
- Addition
- Subtraction
- AND
- OR
- XOR
- NOT

## Tools Used
- Xilinx Vivado
- Verilog HDL

## Author
Ruthvik R

Title: 8-bit ALU design  
 

Description: In the final project, you have to design an 8-bit ALU (Arithmetic Logic Unit). The arithmetic logic unit will perform eight different operations depending on the opcode selected. If we are having eight different operations that means we need three bits for the opcode. The eight different operations that this ALU will perform are as follows:

Addition 
Subtraction 
Multiplication 
Left shift 
Right shift 
Logical AND 
Logical OR 
Logical XOR 

Input cases: 

Set the two inputs as 45 and 38.
Now, set the opcodes as 000, 001, 110, and 111 respectively and find the outputs. 
Set the inputs as 49 and 10 
Now, set the opcodes as 010, 011, 100, and 101 respectively and find the outputs. 
Expected outcomes. 

For the opcode values, the respective operations would be performed and the results should be in as per the arithmetic operations. 
000 - Addition 
001 - Subtraction 
010 - Multiplication 
011 - Left shift 
100 - Right shift 
101 - Logical AND 
110 - Logical OR 
111 - Logical XOR
